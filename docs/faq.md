# FAQ

## Is VELA an app?

No. VELA is a workflow environment for Codex. HELM is the optional app-style local board.

## Do I need HELM?

No. Use VELA directly in your Codex environment. Add HELM later if you want a visual board.

## Does VELA write my paper?

No. VELA helps keep the research path inspectable. Codex can help with bounded tasks, but claims and evidence still need review.

## Can I use my own project structure?

Yes. Keep the core distinctions: materials, evidence, claims, methods, deliverables, and handoffs.

## Can I use Zotero, Obsidian, Python, R, or notebooks?

Yes. VELA records how those tools connect to the project state; it does not replace them.
