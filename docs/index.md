# Codex Research Stack Docs

This documentation site is the public guide to a research workspace for Codex, built for social science projects.

它面向的不是只会看 skill 与 contract 的高级用户，而是希望尽快理解：

- 这是什么
- 适合谁
- 一个研究项目如何在里面推进

## Start Here

- [Getting Started](./getting-started.md): start a project, understand the main workspace, and choose the right first action
- [Use Cases](./use-cases.md): see how literature reviews, platform cases, CSS projects, and writing workflows fit the workspace
- [Integrations](./integrations.md): understand how Zotero, Obsidian, browser-visible evidence, and local tooling fit together
- [Repository and Future App](./app-relationship.md): see what the repository should own and what a future app should own

## For Advanced Users

- [Architecture](./architecture.md): inspect routing, squads, gates, and compatibility layers
- [Operator Guide](./operator-guide.md): learn the internal control surface
- [Roadmap](./roadmap.md): see what the public product is trying to become next

## Main Workspace Model

- **Projects**: project setup, stage flow, milestones, blockers
- **Sources**: literature, websites, datasets, platform evidence
- **Analysis**: methods, squads, extracted claims and notes
- **Writing**: drafts, writing checks, submission readiness
- **Settings**: local setup, integrations, validators, advanced controls
