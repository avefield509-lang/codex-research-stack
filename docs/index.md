# VELA

VELA is a portable research workflow environment for Codex. It gives a project a readable structure before work spreads across files, notes, citations, datasets, and agent handoffs.

## Relationship To HELM

VELA is the workflow environment. HELM is the optional local research board. You can use VELA without HELM; HELM can later make the same project state easier to inspect.
