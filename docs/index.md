# Codex Research Stack Docs

This documentation site is the public guide to a plugin-first research operating layer for Codex.

它不是内部备忘录的直接公开版，而是面向 GitHub 访客重写过的公开说明层。

## Start Here

- [Getting Started](./getting-started.md): install the public repo, inspect the stack, and bootstrap a project
- [Architecture](./architecture.md): understand the control plane, squads, gates, and knowledge handoff
- [Use Cases](./use-cases.md): see how the system behaves in real research scenarios
- [Integrations](./integrations.md): understand Zotero, Obsidian, social evidence, and plugin boundaries
- [Public Boundary](./public-boundary.md): see what is intentionally excluded from the public repo
- [Roadmap](./roadmap.md): see what the stack is trying to become next

## What This Public Repo Shows

- a plugin-facing research entrypoint
- a contract-driven multi-agent orchestration layer
- explicit pipeline gates for research work
- reusable project scaffolding
- evidence-aware integrations rather than generic tool dumping

## What It Does Not Try To Show

- one user’s private workstation
- cloud operator notes
- secret credentials
- local trust state
- personal application materials
