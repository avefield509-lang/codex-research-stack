New Project

Build the evidence path first.

A new VELA project starts with a question, materials, method expectations, evidence rules, and handoff conventions. The visible path matters more than the speed of any generated artifact.

Question

Define the project scope and expected deliverables.

Materials

Capture source clues without overstating their evidentiary status.

Evidence

Verify source, access time, rights, ethics, and support relation.

Handoff

Prepare bounded context for Codex or HELM.
