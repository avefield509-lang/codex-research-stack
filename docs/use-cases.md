# Use Cases

## Literature Review

VELA helps a reviewer keep source records, candidate claims, evidence status, and synthesis readiness separate. The useful output is not a generated review; it is a project state that shows what can be cited, what still needs verification, and what cannot yet support a claim.

## Social-Platform Case Study

VELA records platform-visible materials with access time, permission boundaries, and ethics notes before they are treated as evidence. This keeps observation, interpretation, and claim support separate.

## Computational Social Science

VELA can sit beside notebooks, scripts, and data folders. It does not replace analysis software; it keeps datasets, method assumptions, evidence links, and deliverables legible while Codex handles bounded tasks.

## Thesis Or Paper Preparation

Before drafting, VELA can show whether the project has a clear question, enough materials, verified evidence, supported claims, method notes, and a reproducible handoff path.
