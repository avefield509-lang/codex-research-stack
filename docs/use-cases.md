# Use Cases

## 1. Literature Review

- route to literature workflow
- verify citations before formal use
- organize structured outputs for later writing

## 2. Computational Social Science

- initialize a project scaffold
- select analysis and writing stages
- require stage gates before delivery

## 3. Social Platform Evidence

- prefer browser-visible evidence
- package repeatable capture as structured artifacts
- avoid hidden-field claims

## 4. Writing Before Export

- capture references actually used in the draft
- verify them
- apply writing-quality checks before export
