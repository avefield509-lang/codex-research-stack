Installation

Portable by default.

VELA is being separated from the older public app workspace into a workflow package that can live inside a user's own Codex environment. The current repository hosts public docs, Pages, release-facing workspace, and the shared visual system.

git clone https://github.com/Marcus-AI4SS/codex-research-stack.git vela
cd vela

Base requirement

Keep project files portable and avoid machine-specific paths in public examples.

Optional companion

HELM can read VELA project state, but the workflow environment must remain usable without HELM.
