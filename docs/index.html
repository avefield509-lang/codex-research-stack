Why it feels different from a generic agent repo

Codex-native, not runtime replacement

The stack treats Codex as the runtime entrypoint and makes the research layer visible through a plugin-first public surface.

Inspectability over mystique

The repo favors schemas, canonical files, review mappings, and gates over hidden heuristics or prompt theater.
