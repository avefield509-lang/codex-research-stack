What VELA Gives You

A project structure and working method that stays useful even when no app is installed.

Project Shape

A clear place for questions, materials, evidence, claims, methods, deliverables, and handoffs.

Evidence Discipline

Materials remain drafts until source, access time, verification, and rights or ethics fields are visible.

Codex Handoffs

Tasks sent to Codex include files, constraints, expected output, and known gaps.

Optional Board

HELM can read the same project state, but VELA does not depend on HELM.

Start Here

1. Download

Clone or download the repository and open it in the Codex environment you already use.

2. Create State

Start a project folder, record materials, and upgrade only verified materials into evidence.

3. Hand Off

Ask Codex for bounded work using VELA's task, file, constraint, output, and gap format.

VELA and HELM share a visual language, but each product can stand on its own.
