Codex Research Stack 是一层以插件为中心的研究控制面， 用来处理 route 选择、项目型多智能体编排、研究 pipeline gate， 以及 Zotero、Obsidian、证据链和复现材料之间的清楚交接。

这套系统公开展示的是什么

它不是一个“prompt 集合”，而是一套把研究自动化关键层显式化的工作栈。

它和一般 agent 仓库的差别

不替代 Codex

这套系统把 Codex 保留为运行时入口，把研究层做成一个更可读的 plugin-first 公共表面。

强调 inspectability

仓库更重视 schema、canonical 文件、review mapping 和 gate，而不是把逻辑藏进提示词戏法。
