科研工作流环境

VELA

Versioned Evidence Lifecycle Architecture，面向 Codex 科研工作的可移植环境包。VELA 封装项目结构、证据规则、方法检查点、交付物流和交接上下文；用户不安装本地看板，也能在自己的 Codex 环境中使用。

快速开始 安装说明 可选集成

独立使用，组合后更顺手

VELA 是工作流环境。HELM 是可选本地科研看板。两者都可以独立存在；组合使用时，项目状态、证据、交付物、环境健康和 Codex 交接更容易检查。

VELA · 科研工作流环境

放进用户自己的 Codex 环境中使用。材料、证据、主张、方法、交付物和交接上下文都保留在可移植工作流结构里。

HELM · 本地科研看板

作为独立本地看板读取项目状态、证据、交付物、环境健康和交接准备度；它不是 VELA 的控制器。

分层

材料、证据、主张、方法和产物保持区分，满足复核条件后才进入下一层。

可追踪

证据路径和交接决策在整个生命周期中保持可见。

可移植

工作流面向本地 Codex 环境迁移，不要求绑定某一个 app。

克制

视觉系统使用淡蓝、白色和深海军蓝，接近 iOS 的轻盈表面，而不是命令控制台。

共享视觉语言：分层帆形、证据路径、导航环和柔和波层。
