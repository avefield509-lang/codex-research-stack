New Project Guide

如果明天开一个新项目，先做什么。

这页只保留可执行顺序：建文件夹、初始化项目、先让系统解释 route，再看是否升级成项目型多智能体。

返回中文首页 先看操作指南

最稳的顺序

  1. 建项目文件夹
  2. 初始化项目脚手架
  3. 先让 `research-autopilot` 解释 route
  4. 如果任务变成项目型，再进入 `research-team-orchestrator`
  5. 用状态板和 gate 文件追踪当前阶段

初始化命令

pwsh -ExecutionPolicy Bypass -File ".\scripts\init-research-project.ps1" -Path "<PROJECT_ROOT>"

初始化后先看

  • `AGENTS.md`
  • `research-map.md`
  • `findings-memory.md`
  • `logs/project-state/current.json`
