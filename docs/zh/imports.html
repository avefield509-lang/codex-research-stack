两个产品，显式本地文件。

VELA = Versatile Experiment Lab & Automation。HELM = Hub for Evidence, Logs & Monitoring。两者保持独立，只通过研究项目内可检查的 JSON 包交换状态。

查看契约 打开 HELM

VELA → HELM

vela.project.context.v1 让 HELM 读取路径、状态、证据计数、文件、阻塞和本地检查。

HELM → Codex / VELA

helm.codex.handoff.v1 是可复制的 Codex 说明。只有用户显式保存或导出时，VELA 才存入项目；HELM 不做静默写回。
