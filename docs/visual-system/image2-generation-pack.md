# ChatGPT-image2 Visual Generation Pack

This pack records the VELA visual system so future image work stays consistent with the current pale-blue, white, calm iOS-like direction.

## Public Asset Rule

Generated drafts should not appear in the GitHub root. Use a local, ignored staging folder:

```text
image2-exports/
```

Only approved finals should be committed under:

```text
docs/assets/brand/
```

## Direction

- Pale blue and white surfaces.
- Navy text, graphite secondary text, restrained deep-blue accents.
- Layered sail, evidence trace, navigation rings, soft wave layers.
- No heavy AI glow, robot faces, neural-network decoration, cyberpunk palette, or app-only dashboard metaphor.
- Keep generated images mostly text-free. Add names and labels in README, Pages, or design files instead of relying on model-rendered typography.

## VELA Mark Prompt

Create a clean premium logo symbol for VELA, a portable research workflow environment. Use a translucent layered sail form with a subtle curved evidence path and three small navigation nodes. Pale blue and white, navy detail, soft iOS-like material, generous whitespace, no text, no robot, no chatbot icon, no dark AI glow.

## VELA Brand Board Prompt

Create a refined brand board for VELA, a workflow environment package for research in Codex. Show the layered sail symbol, evidence trace, package touchpoints, motion cues, and guiding principles. Pale blue and white, calm iOS visual language, premium minimal composition, no dense text, no fake UI metrics, no cyberpunk styling.

## Dual-Brand Prompt

Create a dual-brand relationship board for VELA and HELM. VELA is Versioned Evidence Lifecycle Architecture; HELM is Hub for Evidence, Logs & Monitoring. Show two independent products sharing one visual language. VELA uses a layered sail and evidence path; HELM uses a navigation ring and pointer. Pale blue and white, calm, premium, no app-controls-workflow hierarchy, no automation pipeline metaphor.
