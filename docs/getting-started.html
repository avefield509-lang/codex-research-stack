Getting Started

Start with a project state.

Create a real project folder, capture materials, upgrade only verified materials into evidence, and prepare scoped handoffs for Codex. HELM can make this visible in a board, but VELA does not require HELM.

Install notes Back to home

1. Define

Record the project question, scope, source boundaries, and expected deliverables in portable project files.

2. Trace

Separate materials from evidence. Evidence needs source, access time, verification status, and rights or ethics notes.

3. Handoff

Send Codex bounded context: task, relevant files, constraints, expected output, and known gaps.
