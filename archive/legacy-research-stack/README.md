# Legacy Research Stack Archive

This archive preserves early private research automation, harness, Zotero/Obsidian, and multi-agent routing assets that are no longer part of the public VELA runtime surface.

The public VELA product now treats these materials as historical reference only:

- do not import these modules from `scripts/vela.py`;
- do not make starter projects depend on this archive;
- do not describe archived assets as required for VELA installation;
- promote ideas out of this archive only after they have a public VELA schema, CLI command, and test.
