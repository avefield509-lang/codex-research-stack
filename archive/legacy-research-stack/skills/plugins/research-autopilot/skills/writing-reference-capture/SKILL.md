# Writing Reference Capture

Use this skill before a writing deliverable becomes formal.

## Purpose

- identify papers actually used in the draft
- exclude candidate papers that never entered the argument
- verify formal references before export
- prepare downstream library and note synchronization

## Hard Gate

An item only enters the formal writing chain when it is:

1. actually used in the writing
2. verified as a real publication
3. associated with a valid DOI
4. suitable for formal citation

## Output

Return:

- project name and slug
- verified papers
- rejected candidates
- target library or collection choice
- downstream note sync target
