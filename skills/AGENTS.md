# AGENTS

This directory contains public VELA Codex skills.

Skill work must preserve the VELA product boundary:

- VELA prepares bounded research work for Codex.
- VELA does not execute hidden agent loops.
- VELA does not depend on HELM, private MCP routing, or local app state.
- VELA skills must keep materials, evidence, claims, methods, deliverables, and handoffs separate.
- Any formal academic citation in a deliverable must be verified outside the skill before being treated as final.
