# Minimal Project Example

This folder shows what the public scaffold looks like after initialization.

It is meant to answer a simple question for GitHub visitors:

**What does VELA actually create once it stops being a README?**

## Included Files

- `AGENTS.md`
- `research-map.md`
- `findings-memory.md`
- `material-passport.yaml`
- `evidence-ledger.yaml`
- `.codex/agents/`
- `logs/project-state/`
- `logs/quality-gates/`

## Why This Example Matters

A public repo about multi-agent research work becomes much easier to trust once visitors can inspect:

- project memory files
- gate files
- project-state files
- project-level agent definitions

instead of only reading abstract architecture claims.
