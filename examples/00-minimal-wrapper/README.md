# VELA Minimal Wrapper

This example shows the smallest VELA project shape that Codex and HELM can both understand.

Key files:

- `AGENTS.md`: project contract for Codex.
- `.vela/context.json`: machine-readable state for HELM.
- `handoffs/H001.yaml`: bounded task packet.
- `handoffs/H001.prompt.md`: rendered prompt to paste into Codex.
